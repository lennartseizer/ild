Working with R Markdown

A concise primer on authoring rich documents with code, prose, and graphics.

What is R Markdown?

R Markdown is a lightweight authoring format that allows you to blend narrative text, code, and outputs in a single document. It is a popular choice for reproducible reporting, blending the readability of Markdown with the analytical power of R.

ℹ︎Use the Knit button in RStudio to render the document into HTML, PDF, or Word with all code executed along the way.

Embedding Code and Output

Code chunks are wrapped in ```{r} fences inside your .Rmd file. When the document is rendered, the chunk is run and the results appear inline with the prose.

```{r}
summary(cars)
```

The example above produces the following summary statistics for the classic cars dataset:

Variable Min 1st Qu. Median Mean 3rd Qu. Max
speed 4.0 12.0 15.0 15.4 19.0 25.0
dist 2.0 26.0 36.0 43.0 56.0 120.0
